One of the most important features of Raycast is the Action Panel. It lets you find and take any action in the app with just a few keystrokes.

-   Build a habit of using `⌘` `K` to search for an action. See and learn the keyboard shortcut to control Raycast even faster.
-   Hit `↵` to perform the primary action. Always. The action adopts to the current context. Use it to launch apps, open commands or view details of a GitHub pull request.
-   In forms, use `⌘` `↵` to submit your input.

# Keyboard Shortcuts

-   `↵` to perform the primary action in lists
-   `⌘` `↵` to submit forms
-   `⌘` `K` to show more actions
